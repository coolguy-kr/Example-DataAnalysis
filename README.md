# Example-DataAnalysis
파이썬을 이용한 그래프 그리기 예제입니다. 
이 소스코드는 python 3을 기준으로 합니다.

# Dependencies
첨부된 파이썬 소스코드는 matplotlib, numpy. pandas를 의존성 패키지로서 사용합니다.
설치되어있지 않다면 pip를 통하여 설치할 수 있습니다.
